# Instruções de Uso - Painel Solvéria RP (Local)

Para que o sistema funcione corretamente (botões de Ligar/Desligar), você deve rodar o servidor no seu computador.

## 🚀 Como Iniciar

1.  **Abra o terminal** na pasta `site jogar`.
2.  **Rode o servidor**: Digite `python server.py`.
3.  **Acesse o site**: Abra o navegador e digite: 
    👉 **http://localhost:3000**

> [!IMPORTANT]
> **NÃO abra o arquivo index.html clicando nele na pasta**. Se você abrir como `file://...`, o sistema não conseguirá salvar o status no arquivo. Use sempre o endereço `localhost:3000`.
